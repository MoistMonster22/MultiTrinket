## MultiTrinket
I used the custom slots from shudnal-ExtraSlotsCustomSlots-1.0.16 to create more trinket slots but saw all teh adrenaline keeps adding up creating a 200+ adrenaline bar which will never trigger in my lifetime.
So, this mod fixes this by taking the highest adrenaline cost from all the equipped trinkets and triggers all of them when it reaches the threshold.
This is also my first mod so I am quite unfamiliar with debugging and Github repo.
